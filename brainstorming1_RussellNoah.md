List of my favorite apps and for each app, 2-3 features of that app that make it stand out:

# 1. Discord
- Voice & Video Communication: Offers high-quality voice and video chat for gaming and community interaction.
- Customizable Servers: Users can create servers with specific roles, channels, and permissions for different purposes.
- Integration with Games & Apps: Includes features like displaying what game you’re playing, Spotify integration, and other bots for added functionality.

# 2. TikTok
- For You Page (FYP): AI-powered feed that provides a customized experience based on user interaction.
- Duet and Stitch: Features that allow users to interact with others’ content in creative ways, such as creating split-screen videos.
- Interactive Effects: A wide array of augmented reality filters that users can apply to their videos.

# 3. Spotify
- Personalized Playlists: Automatically generated playlists like Discover Weekly tailored to user tastes.
- Cross-Device Syncing: Seamlessly transitions from one device to another while keeping track of listening progress.
- Collaborative Playlists: Allows multiple users to add songs and build playlists together.
